# Accu Darts

Accu Darts is an online scoreboard for darts players.

### Live Version:
https://accudarts.netlify.app/

#### Features:
- Game average
- Checkouts suggestions
- Game Type (301, 501)
- Legs count
- Amount of relevant scores tracking (180s, 140s, etc.)
- Score history

#### Tech used:
- Html
- SASS
- Javascript
- ReactJs
